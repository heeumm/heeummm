# 🕹 Agentic Audit: 다차원 속성 기반 필터 버블 탐지 프로젝트

본 프로젝트는 스마트 TV 플랫폼의 컨텐츠 추천 시스템에서 발생하는 '가짜 다양성' 문제를 해결하기 위해, **LLM 기반의 에이전틱 오딧(Agentic Audit)** 프레임워크를 제안하고 검증합니다.

## 1. 문제 의식: "알고리즘은 우리를 자유롭게 하는가?"
* **과거 포털 뉴스의 확증 편향 연구**에서 확장하여, 스마트 TV 추천 시스템이 유저를 취향의 감옥에 가두고 있지 않은지 진단합니다.
* 단순히 장르만 섞인 리스트가 아닌, **톤(Tone)과 복잡도(Complexity)** 등 보이지 않는 속성까지 다양해야 진정한 추천의 가치가 발생한다는 가설을 세웠습니다.

## 2. 주요 방법론 (Methodology)
* **4D 분석**: 장르, 소재, 톤, 도덕적 복잡도라는 4가지 심층 속성을 추출하여 컨텐츠의 '해상도'를 높였습니다.
* **에이전틱 오딧 (Agentic Audit)**: AI 에이전트가 인간이 설정한 정답(Ground Truth)의 맹점을 스스로 찾아내고 비판적으로 재검토하는 프로세스를 수행합니다.

## 3. 실험 결과 및 성능 지표
Proposed 모델은 기존 Baseline 대비 정밀한 타겟팅 성능을 입증했습니다.

| Metric | Baseline (1D) | **Proposed (4D)** | Improvement |
| :--- | :--- | :--- | :--- |
| **Precision** | 0.11 | **0.20** | **+81% 상승** |
| **Recall** | 0.20 | 0.20 | (상세 분석 참조) |
| **F1-score** | 0.14 | **0.20** | **+42% 상승** |

* **거리 분포 분석**: Proposed 모델은 고분산(High Variance) 특성을 보이며 컨텐츠 간 미세 차이를 명확히 식별해냈습니다.

## 4. 깊은 고찰 (Conclusion & Insights) 💡
본 프로젝트를 통해 기술적 수치를 넘어 추천 시스템의 본질적인 가치를 고민했습니다.

* **Ground Truth의 주관성 교정**: 인간은 '범죄'라는 키워드만 보고 *The Hangover*를 필터 버블로 오인했으나, AI는 '가벼운 톤'을 읽어내어 이를 교정했습니다. 이는 AI가 단순 정답 추론기가 아닌 **인간의 판단 맹점을 감사(Audit)하는 도구**로 진화할 수 있음을 보여줍니다.
* **다양성의 재정의**: 다양성은 고정된 정답이 없는 주관적 영역입니다. 단순히 '다른 장르'를 주는 것이 아니라, **어떤 속성을 유지하고 어떤 속성에 변화를 줄 것인가**에 대한 세밀한 설계가 필요함을 깨달았습니다.
* **분석가로서의 소회**: 정적인 지표(Recall)에 일희일비하기보다, 모델이 데이터의 질감을 얼마나 정밀하게 파악하는지를 오딧(Audit)하는 과정 자체가 진정한 의미의 다양성 연구임을 확인했습니다.
